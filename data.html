The Russian State Duma Legislators, 2000–2016

This data set contains biographical and other information on 1266 legislators who served in the Russian State Duma in 2000–2016. The data set accompanies the following paper:

Shirikov, Anton. 2021. "Who Gets Ahead in Authoritarian Parliaments? The Case of the Russian State Duma." The Journal of Legislative Studies.

The data set is maintained at Harvard Dataverse.
